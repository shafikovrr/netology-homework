# Домашнее задание к занятию "`Название занятия`" - `Фамилия и имя студента`


### Инструкция по выполнению домашнего задания

   1. Сделайте `fork` данного репозитория к себе в Github;
   2. Выполните клонирование данного репозитория к себе на ПК с помощью команды `git clone`; *надо поменять название репо под название занятия?*
   3. Выполните домашнее задание и заполните у себя на ПК этот файл "README.md":
      - впишите вверху название занятия и вашу фамилию и имя
      - в каждом задании добавьте решение в требуемом виде (текст/код/скриншоты/ссылка на другой файл в вашем репозитории)
      - при оформлении используйте возможности языка разметки md (коротко об этом можно посмотреть в [инструкция  по Markdown](https://github.com/netology-code/sys-pattern-homework/blob/main/md-instruction.md))
      - *написать, как заливать скриншоты*
   4. После завершения работы над домашним заданием сделайте коммит (`git commit -m "comment"`) и отправьте его на Github (`git push origin`);
   5. Убедитесь, что ваш репозиторий неприватный;*как им это сделать, давай напишем*
   6. Для проверки домашнего задания преподавателем в личном кабинете прикрепите и отправьте ссылку на решение в виде md-файла в вашем Github.
